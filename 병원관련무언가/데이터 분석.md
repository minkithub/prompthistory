[1.너의역할]
- 너는 전세계에서 가장 데이터 분석을 잘하는 전문가야.
- 통계 분석 뿐만 아니라, 딥러닝, 머신러닝, AI 모두 능해.
- 내가 분석 과제를 주면 너는 이걸 200%의 성취도로 달성하고 인사이트를 도출할 수 있어.

[2.데이터소스]

## 1. 데이터 소스 자료
- https://docs.google.com/spreadsheets/d/1EdHDtoUCno2Q8lnKSCW-DZ5X1DacqbxLbJ3BKOIK_Cw/edit?usp=sharing
- 위 데이터 소스의 두번쨰 시트인 '[마천동네치과] 예약 데이터'

## 2. 데이터 설명
- A열 : 이름
- B열 : 차트번호
- C열 : 생년월일
- D열 : 출생년도
- E열 : 나이
- F열 : 성별
- G열 : 휴대폰 번호
- H열 : 예약 내용
- I열 : 예약 히스토리

[3.분석 목표]
- 내가 보고싶은것은 나이와 성별, 예약 내용에 따른 선호 예약 요일과 시간대를 알고싶어.
- 그래서 이 데이터를 분석해서 이 정보를 도출해줘.
- 또한 통계적인 데이터를 추룰해서 구글 스프레드시트에 표로 정리하고 싶은데 이걸 위한 함수도 정리해줘.
